# Tonic Solfa Studio Templates

This folder contains template files that can be loaded into Tonic Solfa Studio using the "File → Load Template..." menu option.

## Supported Template Types

### Native Templates (.tss, .json)
- **.tss**: Native Tonic Solfa Studio project files
- **.json**: JSON format project files
- These contain complete score data including title, composer, key, time signature, tempo, and musical content

### MusicXML Templates (.xml, .musicxml, .mxl)
- Standard MusicXML format files
- Can be created by other music notation software (MuseScore, Finale, Sibelius, etc.)
- Contains detailed musical notation data

### ABC Notation Templates (.abc)
- Text-based music notation format
- Simple and human-readable
- Good for folk music and basic melodies

### MIDI Templates (.mid, .midi)
- Binary MIDI files containing note and timing data
- Create these by exporting from Tonic Solfa Studio or other music software
- Useful for importing basic musical structures

## How to Use Templates

1. **Load Template**: Use "File → Load Template..." to import any of these files
2. **Save Template**: Use "File → Save as Template..." to save your current work as a reusable template
3. **Create Custom Templates**: Start with a blank score, add your preferred settings (key, time, clef, staff type), then save as template

## Template Examples

- `Blank Piano Template.tss` - Empty piano score (treble + bass staves)
- `Simple Melody Template.json` - Empty single staff melody
- `Basic MusicXML Template.xml` - Simple C major melody in MusicXML format
- `Basic ABC Template.abc` - Simple melody in ABC notation

## Creating Your Own Templates

1. Open Tonic Solfa Studio
2. Create a new score or modify an existing one
3. Set up your preferred:
   - Staff configuration (single, piano, grand, etc.)
   - Key signature
   - Time signature
   - Tempo
   - Clef settings
4. Add any initial notes or structure you want
5. Use "File → Save as Template..." to save it

The template will be saved in this folder and appear in the Load Template dialog.
